# DataNexus Web Interface

This is the frontend for the DataNexus marketplace, built with Next.js, Tailwind CSS, and Framer Motion.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- **Visually Amazing UI**: Modern dark theme with gradients and glassmorphism effects.
- **Marketplace**: Grid view of datasets with filtering (mock).
- **Dataset Details**: Cards showing price, format, size, and tags.
- **Responsive Design**: Optimized for mobile and desktop.

## Tech Stack

- Next.js 15 (App Router)
- Tailwind CSS v4
- Framer Motion
- Lucide React Icons
